Don't Delay

This is a simple one-page web-app that allows users to check if a given airport is delayed or not. It also provides some useful, miscellaneous information such as weather and temperature. It uses the FAA's public flight status API found here

Major Airports

This is a list of some major airports that may come in handy. Instead of having to search for all of the airports manually, this small list has been created to show a select few important airports' conditions. If you would like the list to be updated with newer values then simply refresh the page. Note that since not all airports with delays will appear here, you can use the form below to check on a specific airport and its conditions.
